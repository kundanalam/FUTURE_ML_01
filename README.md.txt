# 📊 Sales Forecasting Dashboard

This project is an interactive **Sales Forecasting Dashboard** built using Machine Learning and deployed with Streamlit.  
It analyzes historical sales data and predicts future sales trends using time-series forecasting.

 🚀 Project Overview
The goal of this project is to:
- Analyze historical sales data
- Forecast future sales using a machine learning model
- Visualize sales trends through an interactive dashboard

Users can upload their sales dataset and generate forecasts to understand future demand patterns.

 🛠 Tools and Technologies
- Python
- Pandas – Data cleaning and manipulation
- NumPy – Numerical operations
- Prophet – Time series forecasting
- Streamlit – Web dashboard interface
- Plotly – Interactive visualizations
- Scikit-learn – Model evaluation metrics
